Merkle tree - Introduction

18 January, 2022

Recently I came across the concept of Merklee tree, and I wanted to learn all about them and how they work. I decided to write a simple article to explain the concept and how to build a Merkle tree the way I grasped it. I will also include a simple implementation of the Merkle tree in C++.

Merkle tree is a tree data structure in which each leaf node is a hash of a block of data, and each non-leaf node is a hash of its children
